## EdTech AI

This project is built with Vite + React + TypeScript and Tailwind CSS.

### Getting started
- Node 20.x
- Install: `npm ci`
- Dev: `npm run dev`
- Lint: `npm run lint`
- Format: `npm run format`
- Test (watch): `npm test`
- Test (CI): `npm run test:ci`
- Build: `npm run build`

### Environment
Copy `.env.example` to `.env` and fill in values.
