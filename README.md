# Authentication (Supabase)\n\nThis app uses Supabase for authentication.\n\n1) Create a .env from the example and fill values:\n\n```bash\ncp .env.example .env\n# set VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY\n```\n\n2) Configure Redirect URLs in Supabase Auth settings:\n- http://localhost:5173 (or your dev URL)\n\n3) Start the app:\n\n```bash\nnpm run dev\n```\n\n4) Visit /auth to sign in via magic link.\n\nNotes:\n- Client is initialized in src/integrations/supabase/client.ts\n- Auth context lives in src/auth/AuthProvider.tsx\n- Use <ProtectedRoute> to guard private pages.\n
