# Contributing

Thanks for your interest in contributing!

- Use Node 20.x
- Install deps: `npm ci`
- Start dev: `npm run dev`
- Lint: `npm run lint`
- Format: `npm run format`
- Test (watch): `npm test`
- Test (CI): `npm run test:ci`

Before committing, Husky will run lint-staged to format and lint changed files.

## Branching
- feature/* for new features
- fix/* for bug fixes
- chore/* for maintenance

## Pull Requests
- Reference related issues
- Ensure CI is green
- Keep changes focused and documented
